# 📚 Web Scraper with Scrapy & MongoDB

A complete web scraping project that extracts book data from books.toscrape.com.

## 🚀 Features
- **Web Scraping**: Automated data extraction using Scrapy
- **Database Storage**: MongoDB integration for data persistence
- **Data Analysis**: Pandas for data manipulation
- **Data Export**: JSON and CSV formats

## 🛠 Tech Stack
- **Python 3.8+**
- **Scrapy** (Web scraping framework)
- **MongoDB** (NoSQL database)
- **Pandas** (Data analysis)

## 📁 Project Structure
web-scraper-project/
├── scraper/           # Scrapy project
├── data/             # Raw and processed data
├── visualization/    # Charts and dashboard
├── requirements.txt  # Dependencies
├── README.md        # Documentation
└── run_scraper.py   # Main execution script

## ⚡ Quick Start
1. Install dependencies:
\\\ash
pip install -r requirements.txt
\\\

2. Run the scraper:
\\\ash
python run_scraper.py
\\\

3. Check the results in \data/\ folder

## 📊 Data Pipeline
1. **Crawling**: Scrapy spider extracts book data
2. **Storage**: Data saved to MongoDB
3. **Processing**: Data cleaned and analyzed
4. **Export**: Data saved to JSON and CSV files
